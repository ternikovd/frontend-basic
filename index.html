• git status — перелік нових або змінених файлів
  • git add "name" — додає бракуючі файли до коміту (?), через "." додає усі файли
  • git commit -am [опис змін] — фіксує зміни та вносить в історію
  • git remote -v — підключення до репозиторієв та гілок
  • git push — завантажує усі локальні зміни до репозиторію
  • git pull — скачує історію з репозиторія
  • git checkout — переміщення між гілками
